Podemos criar também um texto marcado usando a tag < MARK >

Texto grande e pequeno

Estamos criando um texto grande e pequeno. Aqui podemos ver um texto maior, um texto tamanho normal e aqui podemos ver um texto menor

Texto deletado

Podemos marcar um texto como exlcuido para indicar que ele deve ser lido, mas não considerado

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois. Utilizando aqui a tag < INS >, que possui sentido e carrega uma semântica de ênfase.

Mas também podemos utilizar para sublinhar o texto a tag < U >, lembrando que essa tag não carrega semântica, ou seja, não possui sentido, apenas forma.
